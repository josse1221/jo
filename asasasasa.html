HTML (HyperText Markup Language) es el lenguaje estándar para estructurar y presentar el contenido en la web. Sirve para definir la organización y los elementos visuales de una página web mediante etiquetas, que actúan como contenedores de contenido.

HTML fue creado para simplificar la conexión entre documentos en línea y facilitar el acceso a la información a través de enlaces (o hipervínculos), permitiendo la navegación entre páginas y sitios web de manera eficiente. Este lenguaje es esencial porque establece una estructura comprensible para los navegadores, que interpretan y muestran el contenido HTML como texto, imágenes, videos y enlaces interactivos.

Desde sus inicios, HTML ha evolucionado para soportar multimedia, gráficos, y una experiencia de usuario mejorada, adaptándose a las necesidades actuales de la web. En la versión actual, HTML5, incorpora funcionalidades para trabajar mejor en dispositivos móviles y permitir el uso de gráficos y aplicaciones web interactivas sin necesidad de complementos adicionales.
